    i-date, simple JavaScript library for the date formatting and parsing



    Goods:

    — simple syntax;
    — compatibility with all existant frameworks;
    — static syntax (no need to make an example to use IDate object methods.



    Requirements:

    — JavaScript.



    Code example:

    <code>
        <script type="text/javascript" src="i-date.js"></script>
        <script type="text/javascript">
            IDate('Y-m-d'); // 2015-02-25 12:33:11
        </script>
    </code>
